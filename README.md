# WeatherAD — Weather Anomaly Detection

## Run locally
```
pip install -r requirements.txt
python app.py
```
Open: http://localhost:5000

## Deploy to Vercel
1. Push this folder to a GitHub repo
2. Go to vercel.com → Import repo
3. Framework: Other | Root: ./ 
4. Deploy!

## CSV Format
month,temperature,rainfall
Jan,28.5,45
Feb,24.0,30
