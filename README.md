# WeatherAD — Weather Anomaly Detection 🌦️📊

## 📌 Overview

WeatherAD is a web-based application developed to detect anomalies in weather data using CSV input.
Users can upload their own dataset or use a sample dataset to analyze unusual patterns in weather parameters.

---

## 🌐 Live Application

👉 https://che110-weather-ad.onrender.com/

---

## ⚙️ Features

* Upload custom CSV weather dataset
* Load sample dataset instantly
* Detect anomalies in weather data
* Visual and structured output
* Simple and user-friendly interface

---

## 📂 Project Structure

* `app.py` – Main Flask application
* `templates/` – HTML templates for UI
* `static/` – CSS and frontend assets
* `api/` – Backend processing / routes (if applicable)
* `requirements.txt` – Python dependencies
* `.python-version` – Python version config

---

## ▶️ How to Run Locally

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run the application:

   ```
   python app.py
   ```
4. Open in browser:

   ```
   http://127.0.0.1:5000/
   ```
